# __TON 25-COLDQ Submitted__
The codes provided here are corresponding to the numerical simulations in our paper, __J. Wang*__, Y. Liu, W. Qin, W. Bao, J. Guo, and M. Zhou, “Decentralized online learning with hard constraints: A doubly-bounded queue approach,” submitted to _IEEE Transactions on Networking (__TON__)_.

This repository contains code for the four experiments presented in our paper, covering both _Time-Varying Constraints_ and _Fixed Constraints_. The experiments include _Convex Logistic Regression_ and _Non-Convex Neural Network_, with a focus on Communication Efficiency. The corresponding code for each experiment can be found in the folders named accordingly. __For detailed explanations of the code, please refer to the README file within each folder.__
